# 🎨 AI Image Generator

An AI-powered web application that generates high-quality images from text prompts using Stable Diffusion.

## 🚀 Features

* Generate images from text prompts
* Clean and modern UI
* Fast API-based image generation
* Responsive design

## 🛠 Tech Stack

* Next.js / React
* Tailwind CSS
* Hugging Face API (Stable Diffusion)

## ⚙️ Setup

1. Clone the repository:
   git clone https://github.com/Visualeo/Aimage-repo.git

2. Install dependencies:
   npm install

3. Create `.env.local` file:
   HUGGINGFACE_API_KEY=your_api_key_here

4. Run the app:
   npm run dev

## 📸 Usage

* Enter a prompt
* Click generate
* Download your AI-generated image
